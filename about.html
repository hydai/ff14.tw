hydai 和快樂夥伴們一起做的 FF14 小工具集合

專案由來

這個專案其實很簡單，就是 hydai 和朋友們在玩 FF14 的時候，覺得一些小工具很方便，但都是英文的，用起來有點不習慣。於是就想說：「不如自己做一個中文版的吧！」

開始做了之後發現還挺有趣的，就慢慢加了更多工具。沒有什麼遠大的理想，純粹就是為了讓自己和朋友們玩遊戲時更開心一點。

我們在做什麼

說穿了就是把一些實用的 FF14 小工具做成中文版，讓大家用起來順手一點。目標很簡單：好用、免費、中文。

  • 收集各種實用的遊戲小工具
  • 翻譯成繁體中文和台灣玩家習慣的用詞
  • 有 bug 就修，有新想法就加
  • 開源分享，歡迎大家一起玩
  • 順便練習寫程式 😄

未來的話

也沒想太多啦，就是繼續把遊戲玩得開心，順便把工具做得更好用一些。如果有人覺得有幫助，那就太好了！

有想到什麼新的工具或是改進的想法，就加進來試試看。反正是開源的，大家都可以一起來玩。

一起來玩

這個專案放在 GitHub 上面，程式碼都是開源的。如果你也想加個什麼功能，或是發現哪裡有問題，都歡迎來一起搞！不會寫程式也沒關係，有想法或建議都可以提出來。

有興趣的話可以到 GitHub 看看，或是直接丟個 issue 跟我們聊聊！
