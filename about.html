關於 FF14.tw

為台灣冒險者打造的 Final Fantasy XIV 工具集合

🎯 我們的理念

FF14.tw 致力於為台灣的 Final Fantasy XIV 玩家提供實用的遊戲工具。我們相信好的工具能夠提升遊戲體驗，讓冒險者們能夠更專注於享受艾歐澤亞的冒險旅程。

✨ 特色功能

  • 完全開源：所有程式碼都在 GitHub 上公開，歡迎貢獻
  • 永久免費：不含廣告，無需付費，純粹為了社群
  • 響應式設計：支援手機、平板、電腦各種裝置
  • 多語言支援：提供繁體中文、日本語、English

🛠️ 技術架構

本站採用純前端技術開發，不需要後端伺服器，確保快速載入與隱私安全：

  • Vanilla JavaScript - 無框架依賴，輕量快速
  • 響應式 CSS - 適配各種螢幕尺寸
  • GitHub Pages - 穩定可靠的託管服務
  • 第三方 API 整合 - 即時資料查詢
